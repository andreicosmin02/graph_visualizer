# Graph Visualizer

Interactive graph visualization and editor with tkinter + matplotlib.

## Usage

```bash
python3 run.py
```

## Controls

- **Click empty space**: Add node
- **Drag node**: Move
- **Click node → Click node**: Create edge
- **Left-click edge label**: Edit weight/capacity
- **Right-click node/label**: Delete
- **Escape**: Deselect
- **Ctrl+S/O/N**: Save/Open/New

## Features

- Interactive node/edge creation and editing
- Dark theme UI
- JSON persistence
- Supports parallel edges
